# Week 7: Spatial History & Web Mapping

**Due**: Monday, March 9 at 9:00 AM

## Assignment Overview

Create an interactive web map visualizing historical data.

## Files

- `mapping.ipynb` - Notebook for data preparation
- `map.html` - Your final interactive map
- `geodata.geojson` or `geodata.csv` - Your georeferenced data
- `AI-use-log.md` - Document AI assistance

## Requirements

Map must include:
- At least 10 georeferenced points or polygons
- Pop-ups with contextual information
- Base map selection
- Legend explaining symbols/colors
- Brief written interpretation (300-500 words)

## Advanced Options (choose at least one)

- Time-slider showing change over time
- Heatmap or clustering
- Multiple layers users can toggle
- Historical map overlay
- Custom icons or styling

## Tools

Use Leaflet.js or Folium (Python). Templates provided.

## Deliverables

- Jupyter notebook with data preparation
- HTML map file (working, interactive)
- GeoJSON or CSV with your data
- Written interpretation
- AI use log

## Grading Criteria

- Map functionality (30%)
- Historical interpretation (25%)
- Design and usability (25%)
- Documentation (20%)
